# راهنمای کاربری TEOS

## ربات تلگرام
- `/start` – شروع و منوی اصلی
- `/admin` – پنل ادمین (در صورت دسترسی)
- بخش‌ها: موزیک، سرویس‌ها، کیف پول، پشتیبانی

## پنل مدیریت تحت وب
- آدرس: `https://your-domain.com`
- ورود با حساب مالک (پیش‌فرض: admin / admin)
- امکانات: مدیریت کاربران، تراکنش‌ها، آهنگ‌ها، سرویس‌ها، منوها، تنظیمات سیستم

## API
- مستندات Swagger: `/docs`
- GraphQL: `/graphql`

## نصب پلاگین
- از پنل مالک بخش افزونه‌ها فایل `.py` را آپلود کنید.
